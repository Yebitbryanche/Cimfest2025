# Cimfest2025
This project is designed for the purpose of the cimfest hackathon 2025 at Mountain Hotel Buea Cameroon November 17th

# frontend 
move to the frontend directory and type the following
1 npm install -- to install all node modules
2 npm run dev -- to open server

# backend
move to backend directory and do the following in the terminal
1 python -m venv venv
2 activate the venv
3 run pip install -r requirements.txt
4 uvicorn main:app --reload to run backend
